# 更新日志 (Change Log)

本项目的所有重要更新都将记录在此文件中。

## [1.0.2] - 2026-03-09

### 🛠 修复 (Fixes)
- 修复历史图片/媒体加载失败时的重试流程：新增前端 URL 解析重试与后端按消息刷新。
- 优化媒体失败场景：离线时不触发后端重试，降低无效请求与日志刷屏。
- 新增“消息不存在(1200)”不可恢复缓存，避免同一历史消息反复重试。
- 修复“无工作区时保存后端设置失败”问题，自动回退写入用户设置（Global）。
- 强化“清空本地会话缓存”：清空会话/目录/群成员等缓存并阻断当前连接历史回填，主页面立即清空。
- 修复清缓存按钮在当前环境下确认弹窗导致无法执行的问题，改为点击即清空。

## [1.0.1] - 2026-03-09

### ✨ 改进 (Improvements)
- 回复引用消息增强：当被引用内容为图片/视频时，引用区显示缩略图。
- 回复引用消息支持悬停预览：鼠标移入引用缩略图可弹出大图/视频预览（视频保持静音）。
- 回复引用区增加“原消息发送者”显示，便于快速识别是谁发的内容。
- 优化回复引用内容展示：有媒体时默认仅展示缩略图，减少文字噪音。

## [1.0.0] - 2026-03-09

### 🎉 首次发布 (Initial Release)
- 提供 VS Code 侧边栏聊天界面（会话列表 + 二级消息页）。
- 接入 NCat OneBot11，支持私聊/群聊消息收发。
- 支持图片消息、图片悬停预览、视频预览（静音）与表情包面板。
- 支持回复、@、戳一戳、撤回、合并转发预览、JSON 卡片消息发送。
- 支持 NCat 后端托管启动、快速登录、自动重连与日志查看。
- 支持本地会话缓存与历史消息预加载。
